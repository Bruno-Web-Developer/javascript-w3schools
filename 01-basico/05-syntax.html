JavaScript Syntax

JavaScript syntax is the set f rules, how JavaScript programs are constructed: 

// How to create variable:
var x;
var y;

// How to use variables:
x = 5;
y = 6;
let z = x + y;

The JavaScript syntax defines two types of values:

  • Fixed values - called Literals
  • Variable values - called Variables

JavaScript Literals

The two most important syntax rules for fixed values are:

  1. Numbers - are writen with or without decimals

    2. 10.50

    1001


  2. Strings are text, writen within double or single quotes

    3. 'Bruno Corrêa'

    "Bruno Corrêa"

JavaScript Variables

In a programming language, variable are used to store data values.

JavaScript use the keywords var, let, const to declare variables

An equal sign is used to assign values to variables.

In this, x is defined as variable. Then, x is assigned the value 6: 

let x;
x = 6
